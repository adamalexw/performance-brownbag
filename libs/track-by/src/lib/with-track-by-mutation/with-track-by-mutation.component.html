We'll notice that DOM isn't fully reloaded but rather just the attributes

We could choose to either track by the index or a unique identifer on the object

Which property to use depends on the use case.

Index

If the number of items remain the same (sorting)

We are adding items to the bottom

Identifier

If we have a large list (which you shouldn't do anyway)

If we have remove functionality (could be the 1st item upsetting all the indexes)

We are adding an item to the top
